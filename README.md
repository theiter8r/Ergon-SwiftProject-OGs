🧠 Burnout Early Warning System
🚀 Overview

A smart HealthTech app that predicts burnout before it happens by analyzing daily patterns like sleep, mood, screen time, and social activity.

💡 Solution

Users follow a simple:

🌅 Morning check-in (sleep, mood, energy)
🌙 Night check-in (screen time, stress, social activity)

The system tracks trends and detects early warning signs.

<h3> Tech stack </h3>
- React (Frontend)
- Three.js (3D UI / Visualization)
- Node.js + Express (Backend)
- MongoDB (Database)

🧠 Core Idea

“We don’t just track habits — we detect patterns and predict burnout early.”
